An endpoint connector that links a WordPress website to the WPAutoPublisher platform for instant AI content generation.

== Description ==

WPAutoPublisher-Client is a powerful plugin that serves as an endpoint connector, bridging your WordPress website with the WPAutoPublisher platform. This integration facilitates almost instantaneous generation of AI-optimized content tailored to user specifications. The platform incorporates advanced AI models, including ChatGPT4, DALLE, and Stable Diffusion, to deliver high-quality content.

== Installation ==

1. Upload `wpautopublisher-client` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Register on [wpautopublisher.com](https://wpautopublisher.com).
4. Log into your account and create an API key.
5. Use this API key to connect your website to the WPAutoPublisher platform.

== Frequently Asked Questions ==

= How can I use it? =
Install the plugin, register at [wpautopublisher.com](https://wpautopublisher.com), log into your account, and create an API key. This key will connect your website to the platform.

= What type of content can I generate? =
Currently, the platform supports article generation(content & images). Product support will be introduced in version 1.1.0, with future versions bringing in features like social media platform support.

= Can I install and use it with Gutenberg blocks enabled on my site? =
Yes, the plugin is compatible with Gutenberg blocks.

= Can I use it if I use the classic editor? =
Absolutely! The plugin works seamlessly with the classic editor as well.
